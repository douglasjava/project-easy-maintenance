# TASK-030 — Tela de comparação de planos (upgrade page)

## Tipo
Produto / UX / Conversão

## Categoria
Frontend / Produto

## Prioridade
🟡 Médio

## Fase
2 — Pós-lançamento

## Épico
EPIC-006 — Produto SaaS

## Descrição
A tela de billing atual mostra os planos, mas pode não comunicar claramente o valor diferencial de cada um. Uma tela de comparação de planos com features side-by-side, destaques e CTA claros é fundamental para conversão de upgrade autônomo (sem necessitar de vendas).

## Critérios de Aceite
- [ ] Tabela de comparação de planos com features lado a lado
- [ ] Plano atual do usuário destacado visualmente
- [ ] Features bloqueadas no plano atual aparecem com ícone de "locked" e link de upgrade
- [ ] Botão "Fazer upgrade" em destaque para planos superiores
- [ ] Preço mensal e anual comparados com desconto visível (se houver)
- [ ] FAQ sobre billing na mesma página

## Esforço
Médio (1 dia)

## Risco de não fazer
Usuários que atingem limites não têm incentivo visual para upgrade — churn por frustração em vez de conversão.

## Status
Concluído
