## Descrição
Usuários que completam o onboarding chegam ao dashboard sem saber o que fazer a seguir. Um tour guiado interativo de 3-5 passos mostra as principais funcionalidades e leva o usuário ao "aha moment" mais rapidamente.

## Critérios de Aceite
- [x] Tour ativado automaticamente na primeira visita ao dashboard
- [x] Tour mostra: dashboard → criar item → registrar manutenção → configurar norma (4 passos)
- [x] Tour pode ser pulado (botão ✕ no tooltip)
- [x] Tour pode ser relançado (botão "?" fixo no canto inferior direito)
- [x] Checklist de onboarding persistente no dashboard até completion ou dismissal manual

## Biblioteca sugerida
- Shepherd.js (open source, React-compatible)
- Driver.js (mais simples)
